1. Install `HomeBrew`.
   Example:

    ```console
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
    ```

1. Install `python3` using `brew`.
   Example:

    ```console
    brew install python3
    ```
